# Social Profile Intelligence Network (SPIN)



**شبكة استخبارات الملفات الاجتماعية (SPIN)** هي أداة مفتوحة المصدر لجمع المعلومات من منصات التواصل الاجتماعي (فيسبوك حاليًا)، مخصصة للأغراض التعليمية والبحثية والأخلاقية فقط.

**المؤلف:** ABDULLATEF
**الإصدار:** 2.0
**ملاحظة:** هذه الأداة مبنية على فكرة مشروع OSIF.

### الميزات الرئيسية

*   جمع بيانات الأصدقاء.
*   عرض معلومات مفصلة عن صديق معين.
*   استخراج معرفات (IDs) الأصدقاء، وأرقام الهواتف، وعناوين البريد الإلكتروني.
*   ميزات البوت للتحكم الآلي (مثل الحذف الجماعي للأصدقاء).

**Social Profile Intelligence Network (SPIN)** is an open-source information gathering tool for social media platforms (currently Facebook), intended for educational, research, and ethical purposes only.

**Author:** ABDULLATEF
**Version:** 2.0
**Note:** This tool is based on the concept of the OSIF project.

### Key Features

*   Gathering friend data.
*   Displaying detailed information about a specific friend.
*   Dumping friend IDs, phone numbers, and email addresses.
*   Bot features for automated control (e.g., mass unfriend).
